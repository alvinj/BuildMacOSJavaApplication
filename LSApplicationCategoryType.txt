These are `LSApplicationCategoryType` for building MacOS applications. They belong in the application's *Info.plist* file.

From the URL below, “LSApplicationCategoryType (String - macOS) is a string that contains the UTI corresponding to the app’s type. The App Store uses this string to determine the appropriate categorization for the app. Table 2 lists the supported UTIs for apps.”

- https://developer.apple.com/library/content/documentation/General/Reference/InfoPlistKeyReference/Articles/LaunchServicesKeys.html#//apple_ref/doc/uid/TP40009250-SW8
